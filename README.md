# @anio-js-foundation/node-fs-remove-sync

Synchronously remove a path.

```js
import nodeFsRemoveSync from "@anio-js-foundation/node-fs-remove-sync"

console.log(
	nodeFsRemoveSync("examples/dest")
)
```
